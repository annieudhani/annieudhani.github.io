Introduction and Data Definition

International trade has occurred since the earliest civilisations began trading, but in recent years international trade has become increasingly important with a larger share of GDP devoted to exports and imports. Different countries are endowed with different assets and natural resources: land, labor, capital, and technology, etc. Global trade enables different countries to sell the goods, which they can efficiently produce, in different countries thereby making whole supply chain very efficient and cost effective.

I have created this dashboard to study the international trade data for commercial goods using various d3 Visualizaton. The technologies used to create this dashboard are HTML5, CSS3, JavaScript, d3.js. The data used to create this dashboard is sourced from UN Comtrade Database.

A yearly report can be downloaded from UN Comtrade Database website containing monetary value of all Commercial goods exported and imported by a country to/from world. I downloaded four reports from year 2017-2020 and combined them into one csv named "all.csv". "all.csv" will enable us to make almost all the Visualizatons that we have learnt in the class. To create a Force layout graph with countries as nodes and trade value between them as link, I had to download multiple reports from the same source since UN comtrade Database restricts us to download trade of all countries either with the world or five selected countries. I wrote a script in JavaScript language to download the report with 5 countries in every iteration of a loop for every year ranging from 2017-2020.

There are around 35 columns in report but the columns which are of interest to us for this dashboard are listed in below table:

Column Name Description
Trade Flow This column indicates the flow of trade from Reporter country. Possible values are Import or Export in this column
Year This column indicates the year for the reported trade. For our case, we have limited it to 4 values(2017-2020)
Trade Value (US$) This column specifies the value of trade in US dollars
Reporter This column specifies the name of the reporter country
Reporter ISO This column specifies the ISO code of trade reporter country
Partner This column specifies the name of the partner country with whom trade has happened
Partner ISO This column specifies the ISO code of partner country with which trade has happened

Questions to Explore:
  1. Which country is the biggest exporter/importer of the goods? We can make use of doughnut/pie chart to answer this question.
  2. Which countries are the top exporters/importers of the world? We can make use of doughnut/pie chart to answer this question.
  3. How the export/imports has progressed for a country from 2017-2020? Did it increase/declined? Bar chart can be used to visualize and answer this one.
  4. What countries have good bilateral relationship in terms of international trade? We can use Force layout graph to explore the relationship here.
  5. How highest trade doing countries are geographically spread? Are they close to each other? Does distance matter for global trades? We can make use of choropleth to answer this question.
